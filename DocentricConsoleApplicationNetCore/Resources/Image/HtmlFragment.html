  2. The maintenance department has trainers/assessors within the team. The site uses Pegasus software for training records management.

  3. A training needs analysis has been conducted. It has been set up as a role profile for each role and then training competencies are mapped to the role.

  4. Premier has a site induction program for all personnel starting on site. Refresher training and skills re-assessment are conducted for some competencies such as permits. Pegasus is used to schedule refresher training before it expires.

  5. The mine has a specific induction for contractors which is based on the mine’s Health, Safety, and Environmental Management System. This induction is current for 2 years. These are run once per week.
